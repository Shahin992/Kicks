# Kicks Frontend Task Setup

React + Vite project scaffolded for the frontend task with:

- React
- Redux Toolkit + React Redux
- Axios
- TypeScript + JavaScript support
- Tailwind CSS
- MUI (Material UI)

## Run

1. Install dependencies:

```bash
npm install
```

2. Setup environment variables:

```bash
cp .env.example .env
```

3. Start dev server:

```bash
npm run dev
```

4. Build:

```bash
npm run build
```
